# Data

Source datasets for the Leithaland accessibility analysis.

| Folder           | Contents                                         |
|------------------|--------------------------------------------------|
| `boundaries/`    | Study area boundary (leithaland_4326, leithaland_polygons) |
| `destinations/`  | kindergartens.gpkg, schools.gpkg, pt_stops.gpkg  |
| `grids/`        | Origin grids (50m, 100m)                          |
| `networks/`      | bike_network.gpkg, walk_network.gpkg             |
| `osm_networks/`  | Raw OSM network data                             |
| `transit/`       | Transit stops and VOR coverage                   |
